## Usage Examples

### Quick Start
```bash
# Install
pip install -e .

# Clone a website
web-all clone https://example.com -o mysite

# Start web GUI
web-all serve
# Open http://localhost:8000
```

### Python SDK
```python
from web_all import SiteCloner, AIEngine

cloner = SiteCloner(output_dir="./output", depth=3)
await cloner.clone_site("https://example.com")
```

### REST API
```bash
curl -X POST http://localhost:8000/api/v1/clone \
  -H "Content-Type: application/json" \
  -d '{"url": "https://example.com", "mode": "everything"}'
```
